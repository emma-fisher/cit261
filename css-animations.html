Designing, Defining, and Triggering CSS3 Animations without Custom Libraries

Animations in CSS allow you to change the CSS of an element gradually, without needing to use JavaScript.

There are many different properties that go along with CSS animations, including the duration, the delay, and the keyframes.

Below you will see a div changing color and position gradually. This is done without needing to press a button and trigger JavaScript, but runs automatically using CSS. How cool! This animation will run three times before stopping because I specified the iteration amount in my CSS.

Local Storage API, Storing and Retrieving Simple Data, Arrays, Associative Arrays, and Objects

The local storage of the browser allows us to store small pieces of information in key value pairs. This can be useful when trying to store something such as the first or last name of a user. This is helpful because this small piece of information is saved even when the browser is closed, and so can be accessed across multiple browser sessions.

We use the setItem function to add a key value pair to the local storage of the browser.

We use the getItem function to retrieve the data we have stored in local storage. In order to retreive an item, the key of it must be used in the function.

We can also use them removeItem function to remove the data we have stored in local storage. In order to delete an item, the key of it must be used inside the function.

After removing the value, when you click the button to get the value, it should now be blank. That key value pair no longer exists in the browser.

JSON Parse

We use the JSON.parse() function to turn text into a JavaScript object.

BEFORE

{ "name":"John","age":31,"pets":[{ "animal":"dog", "name":"Fido" },{ "animal":"cat", "name":"Felix" },{ "animal":"hamster", "name":"Lightning" }]}

AFTER

It allows us to more easily access information in an object by using the object.key syntax. This method is especially helpful when receiving information from an AJAX request that needs to be formatted in a readable and usable way.

Note that if this is used on a JSON derived from an array, it will return an array rather than an object. Click here for an example

Avoid using functions and dates inside JSON objects.

JSON Stringify

We use the JSON.stringify() function to turn objects into strings. In a way, it is the opposite to JSON.parse().

BEFORE

{ name: "John", age: 30, city: "New York" }

AFTER

Doing this prepares objects to be sent to the server.

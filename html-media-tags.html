HTML5 Tags - Video, Audio, and Canvas

The HTML5 Video Tag allows a video to be played in the browser without needing any plugins as with previous versions of HTML. It appears as an embedded video in the webpage, as shown below.

The text between the opening and closing tags of video, audio, or canvas does not appear in the browser unless it is not supported in that browser.

The HTML5 Audio Tag allows an audio file to be played in the browser, again without needing any plugins as was needed in previous versions of HTML. It is also embedded into the browser, as shown below.

The HTML5 Canvas Tag is used as a container to hold graphics on a webpage. Keep in mind that since this is jsut a container, the graphics must still be drawn using JavaScript apart from using the Canvas tag.

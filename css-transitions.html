Designing, Defining, and Triggering CSS3 Transitions without Custom Libraries

CSS has a transition property that allows you to change property values smoothly and over a certain amount of seconds.

The transition property is shorthand for the four properties in a transition: transition-delay, transition-duration, transition-property, and transition-timing function.

For example, I can change the width of this box when I hover over it. Give it a try!

I can change the color of the box too in a transition when hovered over.

I can also transition more than one thing at a time by specifying 'all' in the transition property.

Object Creation Functions, Inheritance, Properties, Methods, Instantiation

Objects are like variables that can hold more than one piece of data.

In order to create an object in JavaScript, you must give a key and value pair.

var obj = {name: 'Emma', age: '20', major: 'software engineering'};

We can also create functions in JavaScript that can create an object when called. For example, I could create a function that creates a Person object every time it is called. Each of the four attributes are called properties.

I would need to pass in the required information when I call the function, which is all the parameters of the function. Calling the function instantiates a new person using all the data I pass in.

var me = new Person(Emma, Fisher, 2, brown);

We can also add methods, or functions, to these JavaScript 'classes'.

me.fullName = function () { return this.firstname + ' ' + this.lastname; }

Let's say that you want to create another class, Student. A student has all of the same things a Person has, and also has a major. Instead of writing out the entire class again for Student, we can make Student inherit from the Person class.

Standard JavaScript events Including those for Mobile Devices and Animation and Transition Events

There are so many different events in JavaScript, but some of the more common ones are:

  • onclick
  • onchange
  • onmouseover
  • onmouseout
  • onkeydown
  • onload

There are two different ways to use these events. One is to add them as an attribute in an HTML element to trigger a javascript function or other code, and the other is to use them only in the JavaScript. I'll demonstrate both below.

Mobile Device Events

There are four special JavaScript events made to be used on mobile devices or other touchscreen devices. They are:

  • touchstart
  • touchend
  • touchmove
  • touchcancel

Keep in mind, these events only work on touchscreen devices.

Animation and Transition Events
